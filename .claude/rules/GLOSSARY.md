# 📚 Глоссарий терминов

## Material Design 3 (2025)

### 🎨 Основные концепции цветовой системы

**HCT (Hue, Chroma, Tone)** - Цветовая модель Material Design 3, где:
- **Hue (Оттенок)** - базовый цвет на цветовом круге (0-360°)
- **Chroma (Насыщенность)** - интенсивность цвета (0-150+)
- **Tone (Тон)** - яркость цвета (0-100, где 0 = черный, 100 = белый)

**Dynamic Color** - Система адаптивных цветов, которая автоматически генерирует цветовые схемы на основе исходного цвета (seed color).

**Seed Color** - Исходный цвет, используемый для генерации всей цветовой палитры приложения.

**Color Scheme** - Полная цветовая схема, включающая все цветовые роли для светлой и темной тем.

### Core colors
Цвета, на основе которых генерируются все остальные палитры. Они включают в себя:
- **Primary**
- **Secondary**
- **Tertiary**
- **Error**
- **Neutral**
- **Neutral Variant**

### 🎭 Цветовые роли (Color Roles)

**Primary** - Основной цвет бренда, используется для ключевых элементов интерфейса.
**Secondary** - Вторичный цвет, дополняет primary и используется для менее важных элементов.
**Tertiary** - Третичный цвет, используется для акцентов и контрастных элементов.
**Error** - Цвет для отображения ошибок и предупреждений.
**Neutral** - Нейтральные цвета для фонов и текста.
**Neutral Variant** - Вариант нейтральных цветов с легким оттенком primary цвета.


### 📊 Тональные палитры (Tonal Palettes)

- **Tonal Palette** - Набор оттенков одного цвета с различными уровнями тона (0-100).
- **Core Palette** - Основная палитра, содержащая primary, secondary, tertiary, neutral и neutral variant палитры.
- **Extended Palette** - Расширенная палитра с дополнительными цветами.

### 🔄 Состояния взаимодействия (State Layers)

- **Hover State** - Состояние при наведении курсора.
- **Focus State** - Состояние фокуса элемента.
- **Pressed State** - Состояние нажатия.
- **Dragged State** - Состояние перетаскивания.
- **Selected State** - Состояние выбора.
- **Activated State** - Состояние активации.
- **Disabled State** - Состояние отключения.


## Технические термины проекта

### 🏗️ Архитектура
- **MaterialColorGenerator** - Основной класс приложения, содержащий всю логику генерации цветов.
- **Monolithic Architecture** - Монолитная архитектура, где вся логика содержится в одном файле.
**Self-Contained Pattern** - Паттерн самодостаточности, все зависимости инкапсулированы внутри класса.

### 📚 Библиотеки

- **@material/material-color-utilities** - Официальная библиотека Google для работы с цветами Material Design 3.
- **material-color-utilities** - Альтернативное название библиотеки.
- **DynamicScheme** - Класс для создания динамических цветовых схем.
- **MaterialDynamicColors** - Класс, содержащий все цветовые роли Material Design 3.
- **TonalPalette** - Класс для работы с тональными палитрами.
- **Hct** - Класс для работы с цветовой моделью HCT.


 ### 🎨 Расширенные цвета
- **Extended Colors** - Дополнительные пользовательские цвета, не входящие в стандартную схему.
- **Custom Color** - Пользовательский цвет с собственным именем и значением.
- **Color Harmonization** - Процесс гармонизации пользовательских цветов с основной схемой.
